## Practice

Which HTML tags create a paragraph of text?

???

* `<p>Text Here</p>`
* `<para>Text Here</para>`
* `<p>Text Here<p>`
* `</p>Text Here</p>`
* `<paragraph>Text Here</paragraph>`
