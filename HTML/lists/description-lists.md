## Revision

Which HTML elements are used to describe a term or name in a description list

???

* ```
<dl>
  <dt>HTML</dt>
  <dd>Content of web page</dd>
</dl>
```
* ```
<dt>
  <dl>HTML</dl>
  <dd>Content of web page</dd>
</dt>
```
* ```
<ol>
  <dt>HTML</dt>
  <dd>Content of web page</dd>
</ol>
```
* ```
<list>
  <term>HTML</term>
  <desc>Content of web page</desc>
</list>
```
